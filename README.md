# Garage Inventory App

Приложение для учета автозапчастей и жидкостей в гараже.

## Функционал
- **CRUD:** добавление, чтение, редактирование и удаление деталей.
- **Типизация:** разделение объектов на запчасти и технические жидкости.
- **Поиск:** фильтрация списка по названию и OEM-номеру.
- **Persistent Storage:** данные сохраняются в БД Room после выхода.

## Стек
- **Язык:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Архитектура:** MVI
- **База данных:** Room
- **DI:** Koin

## Как запустить
1. Откройте папку проекта в **Android Studio**.
2. Дождитесь завершения синхронизации Gradle.
3. Выберите эмулятор или подключите реальное устройство.
4. Нажмите **Run** (зеленый треугольник).
